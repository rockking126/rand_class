# 中小学入学分班系统
中小学入学分班系统；
通过随机分配的方式将学生分配到指定的班级数量中，自动平衡各班人数，自动平衡各班男女比例；
过程中实时显示随机过程，展示随机抽取信息的信息详情；
可以通过excel 导入名单；
可以导出excel 格式的名单；
已于2020年8月在昆明市某学校进行实际使用，效果稳定。
